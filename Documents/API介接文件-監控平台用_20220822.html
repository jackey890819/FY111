上傳流程

開始操作時，請先更新操作紀錄，再上傳成績&職災&關卡，有更新操作紀錄後，就可以進行成績&職災&關卡的單一測試及更新了!!

再次更新操作紀錄，則為新的操作紀錄，即之後上傳的成績&職災&關卡皆屬於該操作紀錄!!

更新操作紀錄

[POST] /api/operationRecord

Request

body

{
   "unit_code": "HA_S"
}

comments

{
   "unit_code": "(課程代碼)"
}

Response 201 建立成功

{
    "success": "Create successfully"
}

Response 400 Bad Request

{
    "errors": "Record failed"
}

Response 401 Unauthorized

{
    "errors": "Unauthorized"
}
